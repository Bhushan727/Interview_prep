Q 1. Difference b/w div and span ?

Ans. div is something we can call as division i.e, it stands for division.
div is an block level element. it occupies the whole available width in the page.it always starts from a new line and leaves a line after by default.
span is an inline element . it covers up only that much of width which is required. it doesn't changes line and doesn't changes line defaultly.

Q 2. Difference b/w class and ids ?

Ans. In HTML class is type of selector.In this type of selector we can use same property for one or more element at a time and can give different class name or same class name .
but, ids are type of selector which is to be defined uniquly i.e, for every element we have to assign different ids in the whole page.

Q 3. What are self-closing tags ?

Ans. In HTML , there are some types of tags which doesn't need a closing tag. these special types of tags are called Self-closing tags. And elements that doesn't have any content in it are called empty tags.
eg. img, br, hr, ....

Q 4. How many types of headings are there? How to decide which to use where?

Ans. In HTML, there are six types of headings which are h1,h2,h3,h4,h5,h6. in all these headings h1 has highest priority and h6 has the lowest priority. The priority order of all these headings are :- h1 > h2 > h3 > h4 > h5 > h6 .
when we have to give heading of top most priority then we should use h1 heading. and when to give lower priority heading we should use h2 heading and so on...

Q5. Difference b/w the block and inline element?

Ans. A block level element is that type of element which always starts from a new line and occupies the whole available width in webpage defaultly. such as . div, h1 to h6, header, footer, table, form, ol, ul etc.. While the inline element is that element which only takes that much of width which is only required for the content and starts within the same line. for instance, span tag,a tag, img, etc..

Q6. Difference b/w em and strong ?

Ans. In HTML, em and strong tags are used for text emphasized.
strong tag is used for making the content strong and add the feelings of importance. it shows the user that content written in strong are urgent. we can use strong for this purpose.
while em tag emphasized content written in it. em tag makes the text look beautiful.

Q7. What are Atrributes ?

Ans. In HTML, Atrributes are those special type of words which add some property or functionality to a paritcular tag. it always written in opening tag and should be written in always lower case because it is case sensitive. we can add multiple attribute to one element.

Ans. In HTML, tags are something we write in between "<" and ">" are called tags "<.tag.>" . while the elemets are opening tag and closing tag and content written withiin it.

Q9. Explain br tag ?

Ans. In HTML, br tag is used to changing the line or line-break. br tag is a self-closing tag i.e, it need not to be closed. we should not use this br tag much , inspite we should use css properties for changing the lines.

Q10. What are Semantic tags ?
