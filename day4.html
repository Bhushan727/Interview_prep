Q8. What is box-sizing?

Ans. In CSS, the box-sizing property defines how much width and height of an element to be calculated. should they include padding and border in actual size or not. by default if we give padding and border width, it actually increases the size of element.

Q9. What is animation-delay ?

Ans. In css, animation-delay property specifies the delay in starting an animation. The value defines in animation-delay are in seconds or milisecond.

Q11. Which property you will use to merge cell horizontally in a grid ?

Q12. Explain the CSS box model .

Ans. Every HTML element is treated as a box which includes content,padding , border and margin , this is known as css box model.
content: this is the actual content of element which can be any text or image or just white space.
padding: the transparent space around content part and inside the border of css box model is the padding area of element .
border: the border line which is around the element that covers up the content is border . it can be of different width and color by giving the property.
margin : The white space around the border of element is the margin area . it is transparent white space . it can be four side or any given particular side.

Q13. What is the diffenence between display none and visibility hidden?

Ans. In CSS, display none and visibility hidden both property is used to vanish the content from browser . but the only diffenence between both of them is visibility hidden hides the element but their occupied space remains and in the case of display none it hides the element as well as distroys the space occupied by it.
