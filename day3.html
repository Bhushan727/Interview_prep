Q2. Explain inline ,internal and external styleSheet

Ans. inline styleSheet : inline css is type of styling in css, where we defiine css in single line within opening tag of html. The major draw for this type of styling is we have to define css seperately for all tags. but this type of css helps us while debugging or checking the code.
internal StyleSheet: internal css is when we define styles in style tag within the same html page in head section. this type of css has less priority than inline. it is used when we don't want to create new css page so we define in the same html page.
external StyleSheet : external type of css is most commonly and professionly used type of css.it works by linking a .css file to the html page. it is less complicated to handle the code in exteranl css.

Q6. What is the z-index in CSS ?

Ans. In CSS, while designing the webpage we have to align items in stack order , this is where the z-index property works for, basically z-index property in css is used for specifying the elements in stack order. higher the z-index value of an element ,higher will be there stack . z-index of an can be negative and positive. the default z-index value of an element is zero(0) .

Q7. What is the difference between padding and margin ?

Ans. In HTML, every element is treated as a box. After these content and border things there are two more things in box model, they are , margin and padding.
Margin is the space around outside of the border and padding is the space between border and the content. it can be all four sides or on bottom, top, left and right only.
